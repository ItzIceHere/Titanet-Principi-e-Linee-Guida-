# Durante il Controllo: Best Practice

Una volta stabilita la connessione, l'indagine deve essere condotta in modo metodico e rispettoso.

1. **Stabilisci il Controllo e Verifica l'Identità:** Una volta connesso, chiedi gentilmente al player di non usare mouse e tastiera. Verifica che l'utente corrisponda alla persona che stai controllando.
2. **Ambiente di Controllo Sicuro:** Chiedi al player di chiudere tutti i programmi non essenziali (browser, client di altri giochi, etc.) per ridurre le distrazioni e le possibili interferenze.
3.  **Segui i Protocolli Tecnici:** Conduci l'indagine seguendo le procedure tecniche standard approvate da Titanet. Questo include l'analisi degli artefatti di sistema, l'uso degli strumenti e degli script autorizzati, e l'ispezione delle cartelle di gioco.

    > Le tecniche specifiche di analisi sono dettagliate nelle guide di formazione interne. Questa sezione descrive il "come" procedere, non il "cosa" cercare nel dettaglio.
4. **Mantieni una Comunicazione Aperta:** Durante il controllo, mantieni il player informato sulle tue azioni, specialmente se stai per eseguire uno script o un tool. Una breve frase come: "Ora eseguo uno tool per verificare le firme digitali dei file recenti" è sufficiente a mantenere la trasparenza.
5. **Rispetta la Privacy del Player:** Durante tutta l'analisi, attieniti scrupolosamente alle regole sulla privacy definite nel **Codice Etico**. Evita attivamente di accedere a cartelle o file personali. Se involontariamente apri un file sensibile, chiudilo immediatamente senza analizzarlo.
