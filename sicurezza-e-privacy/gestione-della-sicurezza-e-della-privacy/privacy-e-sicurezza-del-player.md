# Privacy e Sicurezza del Player

Il player ci concede un accesso temporaneo e limitato al suo sistema. Questo privilegio deve essere trattato con il massimo rigore etico. Il nostro mandato è cercare prove di cheating, non curiosare nella vita privata di una persona.

**Regola Fondamentale: Il Principio del Minimo Accesso** Accedi solo a ciò che è **strettamente necessario** per l'indagine. Se non hai un motivo fondato e collegato alla ricerca di cheat, non hai il permesso di aprire un file o una cartella.

**Aree e File Assolutamente Vietati**

L'accesso alle seguenti aree è **severamente proibito**, a meno che non ci sia un sospetto diretto e documentabile che contengano prove. Anche in quel caso, procedi con cautela e, se possibile, chiedi al player di aprire il file per te.

* ❌ **Cartelle di Documenti Personali:** `Documenti`, `Immagini`, `Video` (limitarsi a file sospetti, non aprire file personali).
* ❌ **Dati di Browser:** Cronologia di navigazione, password salvate, cookie, preferiti. L'unica eccezione è la cronologia dei download, da analizzare in modo mirato.
* ❌ **Applicazioni di Messaggistica:** Non leggere mai le chat di Discord, Telegram, WhatsApp Web, o qualsiasi altra applicazione di comunicazione.
* ❌ **Password Manager e File di Testo con Credenziali:** Non aprire mai file che sembrano contenere password, dati di accesso o altre informazioni sensibili.
* ❌ **Software Finanziari o Bancari.**
* ❌ **Posta Elettronica (Email).**

**Comportamenti Obbligatori**

* **Non Modificare o Cancellare File del Player:** Il tuo ruolo è di osservare. Non devi rinominare, modificare, spostare o cancellare alcun file che non sia stato creato da te durante il controllo (es. file di log di un tool). L'unica eccezione è se stai ripristinando un file da un tool di recupero, e in tal caso devi comunicarlo chiaramente.
* **Non Scaricare File Personali:** È assolutamente vietato trasferire qualsiasi file dal computer del player al tuo, a meno che non si tratti di una prova diretta (es. il file di un cheat), e solo dopo aver informato il player.
* **Comunica le Tue Azioni:** Se devi accedere a una cartella che potrebbe contenere file personali (es. `Download`), informa il player del motivo: "Ora controllo la cartella dei download per verificare file recenti, ignorerò qualsiasi file non pertinente".
