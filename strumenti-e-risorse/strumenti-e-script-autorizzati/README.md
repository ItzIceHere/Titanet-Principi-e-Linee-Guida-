# Strumenti e Script Autorizzati

Per garantire la sicurezza, l'affidabilità e la coerenza di ogni screenshare, il team di Titanet adotta una lista ufficiale e tassativa di strumenti e script che possono essere utilizzati durante un controllo. L'uso di qualsiasi software non presente in questa lista è **severamente proibito**, a meno che non venga esplicitamente autorizzato dal SS Manager in circostanze eccezionali.

Questa politica serve a:

* **Proteggere il Player:** Utilizziamo solo software testato e verificato, privo di malware o funzionalità invasive.
* **Standardizzare le Procedure:** Assicura che ogni staffer utilizzi gli stessi strumenti, rendendo i controlli più equi e i risultati più consistenti.
* **Garantire la Validità delle Prove:** Le prove raccolte con strumenti non autorizzati potrebbero essere considerate non valide in caso di appello.
